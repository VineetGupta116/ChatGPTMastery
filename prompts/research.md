# Research Prompts

> Domain: Deep Research, Competitive Intelligence, Industry Analysis

---

## Senior Analyst Research
```
Act as a senior [domain] analyst. Research [topic].
Structure output as:
1. Executive Summary (3 sentences)
2. Key Findings (numbered, max 7)
3. Implications for [business/client]
4. Open Questions
5. Recommended Next Steps

Use Extended thinking effort. Cite sources inline.
```

## Competitive Intelligence
```
Research [competitor/brand] across web and recent news.
Output:
- Positioning statement
- Key differentiators (vs [my brand])
- Recent moves (last 90 days)
- Weaknesses to exploit
- Threats to monitor

Be direct. No fluff.
```

## Deep Research with Source Pinning
```
Research [topic].
In the research plan:
- Pin sources: [URL1], [URL2], [URL3]
- Allow general web for gaps

Output: Full report with inline citations, executive summary, and 3 recommended actions.
```

## Market Landscape Scan
```
Map the [industry] landscape for [year].
Cover: market size, top 5 players, emerging challengers,
key trends, regulatory shifts, and whitespace opportunities.
Format as a structured brief.
```

## Pulse Task (Daily Briefing)
```
Every morning at 8am IST:
Research latest developments in [industry/niche].
Summarize top 3 signals.
Flag anything actionable for [business context].
Keep it under 200 words.
```
