# Analysis & Strategy Prompts

> Domain: Business Strategy, Data Analysis, Decision Frameworks

---

## Strategic Options Analysis
```
I'm deciding between [Option A] and [Option B] for [context].
Evaluate both using:
- First principles
- Short-term vs long-term trade-offs
- Resource requirements
- Risk profile
- Reversibility

Give a clear recommendation with reasoning.
```

## Data Interpretation
```
Analyze this data and tell me:
1. What is actually happening (not just what the numbers say)
2. What is surprising or anomalous
3. What is missing from this data
4. What decision this data supports

[paste data or upload file]
```

## First Principles Breakdown
```
Break down [concept/problem] from first principles.
Do not use analogies as primary explanation.
Identify: core components, underlying mechanics, failure modes.
End with: what most people get wrong about this.
```

## SWOT + Strategic Implication
```
Run a SWOT analysis for [company/project/idea].
After the SWOT, synthesize the top 3 strategic implications
(not just observations). What should we actually do?
```

## Prompt Chain for Complex Decisions
```
Step 1: "Map all the variables involved in [decision]."
Step 2: "Rank them by impact and uncertainty."
Step 3: "For the top 3 uncertainties, what would change our decision if resolved?"
Step 4: "Given this, what is the minimum viable next action?"
```
