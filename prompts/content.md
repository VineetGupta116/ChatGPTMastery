# Content & Copywriting Prompts

> Domain: Brand Voice, Social, Long-form, SEO

---

## Brand Voice Draft
```
Write a [content type] for [brand/client].
Tone: [Professional / Friendly / Candid / Authoritative]
Audience: [describe]
Length: [X words]
CTA: [what action should reader take]

Brand personality: [describe in 2 sentences]
```

## SEO Blog Post
```
Write an SEO-optimized blog post on [topic].
Target keyword: [keyword]
Secondary keywords: [list]
Length: [800/1200/2000] words
Structure: H1 > H2 sections > FAQ > CTA
Tone: [describe]
Do not use filler phrases. Every sentence must add value.
```

## LinkedIn Post (Thought Leadership)
```
Write a LinkedIn post on [topic/insight].
Style: First-person, punchy, no hashtag spam.
Hook: Start with a counterintuitive statement.
Length: 150-200 words.
End with a question to drive comments.
```

## Email Sequence (3-part)
```
Write a 3-email nurture sequence for [product/service].
Audience: [describe]
Goal: [awareness / conversion / re-engagement]
Email 1: Problem awareness
Email 2: Solution framing
Email 3: CTA with urgency
Tone: [describe]
```

## Website Copy (Section)
```
Write copy for the [Hero / About / Services / FAQ] section
of [brand name]'s website.
Core value proposition: [describe]
Target customer: [describe]
Tone: [describe]
Length: [X words max per section]
```
