# Prompt Library

Reusable, domain-organized prompt templates for ChatGPT (March 2026).

## Files

| File | Domain | Key Use Cases |
|---|---|---|
| [research.md](research.md) | Research & Intelligence | Deep research, competitive analysis, daily briefings |
| [coding.md](coding.md) | Code & Engineering | Codex agents, debugging, architecture, API integration |
| [content.md](content.md) | Content & Copywriting | Brand voice, SEO, LinkedIn, email, web copy |
| [analysis.md](analysis.md) | Analysis & Strategy | Strategic decisions, data interpretation, first principles |
| [ai-systems.md](ai-systems.md) | AI Systems & Prompting | System prompts, prompt eval, multi-agent design, CoT |

## Usage Notes
- Replace `[bracketed]` placeholders with your specifics
- Pair with correct model tier: Instant for drafts, Thinking for reasoning, Pro for high-stakes
- Use inside ChatGPT Projects for persistent context injection
- Combine templates via prompt chaining for complex workflows
