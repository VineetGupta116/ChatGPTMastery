# WOW Creations — ChatGPT Project Setup

## Project Name
`WOW Creations | Website & Digital Presence`

## Custom Instructions (paste into Project Instructions)
```
You are an AI assistant supporting WOW Creations' digital presence build.
Focus areas: website copy, branding, SEO, social content, and UX recommendations.
Default tone: creative, energetic, conversion-focused.
Always align outputs with the brand's visual identity and voice guidelines.
When in doubt, prioritize clarity and conversion over cleverness.
```

## Recommended Files to Upload
- Brand identity guide (colors, fonts, logo usage)
- Website wireframes or sitemap
- Existing copy drafts
- Target audience personas
- Product/service catalogue
- Competitor website screenshots or notes

## Key Prompt Shortcuts

### Homepage Hero Copy
```
Write hero section copy for WOW Creations.
H1: [attention-grabbing headline]
Subheadline: [value proposition, max 15 words]
CTA button: [action word]
Tone: bold, modern, conversion-focused.
```

### Social Media Calendar
```
Create a 2-week social media content calendar for WOW Creations.
Platforms: [Instagram / LinkedIn / X]
Themes: [brand awareness / product showcase / testimonials]
Format: Date | Platform | Caption | Hashtags | Visual suggestion
```

### SEO Audit Response
```
Based on the SEO audit uploaded, prioritize the top 5 fixes
for WOW Creations' website. Order by impact vs effort.
Output: Fix | Why It Matters | How to Implement | Priority
```

### UX Copy Review
```
Review the uploaded page copy for WOW Creations.
Flag: unclear CTAs, weak headlines, friction points.
Suggest rewrites for each flagged item.
```
