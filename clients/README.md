# Client Project Templates

Per-client ChatGPT Project setup guides for consistent, context-rich AI-assisted delivery.

## Structure

Each client folder contains:
- `project-setup.md` — ChatGPT Project configuration instructions
- `custom-instructions.md` — Custom instructions to load per client
- `file-index.md` — Recommended files to upload to the Project
- `prompt-index.md` — Client-specific prompt shortcuts

## How to Use
1. Create a new ChatGPT Project named after the client
2. Load `custom-instructions.md` into Project Instructions
3. Upload all files listed in `file-index.md`
4. Connect Slack/Drive sources if applicable
5. Use `prompt-index.md` shortcuts inside the Project
